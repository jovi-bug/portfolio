# Personal Portfolio - Tjorve Nielsen

A modern, responsive portfolio project developed with React and Tailwind CSS. This portfolio showcases my projects, tech stack, and journey as a web developer.

## Features

- **Responsive Design:** Optimized for all devices (Mobile, Tablet, Desktop).
- **Internationalization:** Support for German and English via `react-i18next`.
- **Animations:** Smooth transitions and interactive elements using `framer-motion`.
- **Modern UI:** Styled with Tailwind CSS (v4) and Lucide Icons.
- **Contact Form:** Integrated contact option via EmailJS.

## Tech Stack

- **Frontend:** React 19, JavaScript
- **Styling:** Tailwind CSS, Framer Motion
- **Icons:** Lucide React
- **Internationalization:** i18next
- **Build Tool:** Vite
- **Deployment:** Vercel

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
